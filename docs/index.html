Canvas LMS UI Component Library

This library provides reusable code snippets for building accessible, responsive UI components in Canvas LMS. All components use inline CSS only (no external stylesheets or JavaScript) and follow WCAG 2.2 accessibility standards.

Accordions

Accordions allow you to present content in collapsible sections, making pages more manageable and organized. These accordions use the native HTML <details> and <summary> elements for built-in functionality without JavaScript.

 Example Accordion Section

This is the content inside the accordion. You can put any HTML content here, including text, images, lists, tables, and more.

<div style="max-width: 100%; margin: 20px auto;">
    <details style="border: 1px solid #d1d9e6; border-radius: 8px; margin-bottom: 16px; overflow: hidden;">
        <summary style="padding: 16px; padding-left: 20px; background: linear-gradient(135deg, #0A1E3F, #112E55); color: #ffffff; font-size: 18px; cursor: pointer; border-left: 4px solid #2CA4CD;" role="button" aria-expanded="false" aria-controls="section-content">&nbsp;Accordion Title</summary>
        <div id="section-content" style="padding: 20px; background-color: #f8f9fa; color: #333333; border-top: 1px solid #d1d9e6;">
            <p style="font-size: 16px; line-height: 1.6; margin-top: 0;">Accordion content goes here.</p>
        </div>
    </details>
</div>
